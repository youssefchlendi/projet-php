
# CoolFood

Our php endOf3rdSem project, To run this project you simply need to create a db named food, import food.sql file.

## Documentation

* UserSide:
    - User can create/update an account
    - User can pass an order from available products
    - User can check his old orders
* AdminSide:
    - Admin can check vital data in dashboard
        - Example : best selling products and best customer
    - Admin can check customer INFOS
    - Admin can approve/disapprove an order and check it's content
    - Admin can Create/Update/Deleate a category
    - Admin can Create/Update/Deleate a product
